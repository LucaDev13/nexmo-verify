Nexmo Verify
================

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

This application requires:

- Ruby 2.2.0 - 2.3.0
- Rails 4.2.5.2


Getting Started
---------------
This application implements Devise authentication in combination with Nexmo Verify API to accomplish two factor authentication
via SMS with authentication code sent to users.
